🌱 The name of my project "Keen-Keeper"



📌 Project Description

Keen-Keeper is a simple and clean web application designed to help users maintain meaningful relationships. It allows users to track interactions with friends, set communication goals, and stay connected through organized timelines and analytics.


🚀 Technologies Used
React.js,
React Router DOM,
Tailwind CSS,
Recharts,
React Icons,
React Toastify,
Daisy UI

✨ Key Features
1. 📊 Friendship Analytics

Visualize your interactions (Call, Text, Video) using a responsive pie chart for better insights.

2. 📜 Timeline Tracking

Automatically logs every interaction and displays them in a clean, filterable timeline.

3. 👥 Friend Management

View all friends, check their contact status, and access detailed profiles with goals and actions.
